## Raspberry-Pi-Controller

Custom Raspberry Pi Controller

## Features

- GPIO* Manager
- File Manager 
    - Upload
    - Download
    - Delete
    - Rename
- SSH Shell (IN PROGRESS)
- CPU, RAM, and Disk Metrics (MORE TO BE ADDED)
- Supports up to 8 Systems (STILL IN TESTING)
- GUI Interface
