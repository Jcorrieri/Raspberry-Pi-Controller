## Raspberry-Pi-Controller

Custom Raspberry Pi Controller

## Features

- GPIO Manager
    - Currently cannot change GPIO configurations  
- File Manager 
    - Upload
    - Download
    - Delete
    - Rename
- Secure Shell
- CPU, RAM, and Disk Metrics
- GUI Interface
- Save System Profile
